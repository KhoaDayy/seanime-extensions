# Quick Start Guide

## 🚀 Cài Đặt Nhanh

### 1. Tải Extension
```bash
git clone <your-repo-url>
cd seanime-extensions
```

### 2. Cài trong Seanime
1. Mở Seanime → **Settings** → **Extensions**
2. Click **Install Extension**
3. Chọn thư mục `seanime-extensions`
4. Enable extension

### 3. Sử Dụng
1. Vào anime page bất kỳ
2. Tab **Online Streaming**
3. Select **AniMapper AnimeVietSub**
4. Chọn episode và enjoy! 🎬

## 📝 Key Points

### ANIMEVIETSUB Only
Extension này chỉ support **ANIMEVIETSUB** provider vì:
- ✅ Nhiều anime nhất
- ✅ Server ổn định (DU/AnimeVsub)
- ✅ Phụ đề VN chất lượng cao
- ❌ ANIMETVN không hoạt động stable

### Server: AnimeVsub (DU)
- Default server Auto-selected
- HLS (M3U8) streaming
- Tốc độ tốt, ít buffering

### Features Highlight
- 📄 **Pagination**: Load tất cả episodes tự động
- 🔄 **Deduplication**: Loại bỏ episodes trùng
- 📊 **Smart Sort**: Hỗ trợ special episodes
- ⚡ **Fast Search**: Direct metadata check

## 🐛 Common Issues

### "No results found"
→ Thử search với tên tiếng Anh
→ Check xem anime có trên AniMapper không

### "No episodes found"
→ Provider chưa có anime này
→ Thử anime khác

### Video không load
→ Check internet connection
→ Verify api.animapper.net accessible
