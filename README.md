# Seanime Vietnamese Extensions

Online streaming provider extensions for [Seanime](https://seanime.rahim.app/) to provide anime streaming with Vietnamese subtitles.

## Supported Providers

| Provider | API Bridge | Status | Servers | Description |
|----------|------------|--------|---------|-------------|
| **AnimeVietSub (Hasukatsu)** | [Hasukatsu API](https://api.hasukatsu.site) | âœ… Available | DU (HLS), FB, VIP | New, optimized version using Hasukatsu API with direct AniList ID mapping. |

## Extensions Installation

1. Open Seanime
2. Go to **Settings > Extensions**
3. Paste one of the manifest URLs below and click install:

### AnimeVietSub (Hasukatsu API - Recommended)
```text
https://raw.githubusercontent.com/KhoaDayy/seanime-extensions/main/animevietsub/animevietsub.json
```
